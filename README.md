PROJECT: Student & Teacher Registration System

DESCRIPTION:
This Java web project allows registration for both Students and Teachers using Servlets and MySQL.
- Student details: Name, Age, Course, Qualification, Location
- Teacher details: Name, Age, Salary, Experience, Specialization, Project

TECHNOLOGIES USED:
- Java (Servlets)
- MySQL Database
- Apache Tomcat Server
- HTML, CSS

HOW TO RUN:
1. Import this project in Eclipse as a Dynamic Web Project.
2. Configure Apache Tomcat and MySQL connection.
3. Create the database 'registrationdb' in MySQL.
4. Run Registration.html → fill details → data will be stored in DB.
5. ViewRecordServlet will display stored records.
